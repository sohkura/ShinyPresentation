Introduction

A Shiny application was developed and deployed to the Shiny server in the following:

https://sohkura.shinyapps.io/ShinyApplication/

This tool provide:

  1. Scatter plot 'Mile Per Gallon' against other parameters
  2. Provide insigts using Color for the plot
  3. Draw Regression Line
  4. Manipulate the MPG mean and calcuate the Mean Squared Error

You can find he source code in https://github.com/sohkura/ShinyApplication.

Operations

In order to plot the data interactively, the following operatoins are provided:

  • Select the X Axies from one of the followings items from the drop down menu:
##  [1] "mpg"  "cyl"  "disp" "hp"   "drat" "wt"   "qsec" "vs"   "am"   "gear"
## [11] "carb"
  • Select the colors from one of the following items from the drop down menu
##  [1] "mpg"  "cyl"  "disp" "hp"   "drat" "wt"   "qsec" "vs"   "am"   "gear"
## [11] "carb"

  • Click Regression Line check box to draw a regression line

  • Move cursor back and forth to specify the mean of the mpg and calcunate the mean squared error
